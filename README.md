# SUV Model Predictor Web App

This is a Django web application for serving predictions from the Survival Analysis model.

## Setup

1.  **Install Dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

2.  **Place Model Files**:
    Copy your trained model artifacts into `predictor/model_files/`. You need:
    -   `config.json`
    -   `encoders.pkl`
    -   `x_mapper.pkl`
    -   `coxph_model.pkl` (or `classification_model.pkl` / `regression_model.pkl` depending on your target)

    These files are generated by the training script in the output directory.

3.  **Run the Server**:
    ```bash
    python manage.py runserver
    ```

4.  **Access**:
    Open your browser and go to `http://127.0.0.1:8000/`.

## Structure

-   `predictor/`: Main Django app.
    -   `utils/model_loader.py`: Logic for loading the model and running inference.
    -   `utils/data/`: Data processing code (from training script).
    -   `utils/models/`: Model definition code (from training script).
    -   `views.py`: Handles web requests.
    -   `templates/`: HTML files.
